# 🗄️ Projetos - Banco de Dados

Este repositório contém atividades, exercícios e projetos desenvolvidos durante a disciplina de **Banco de Dados**.

O objetivo é praticar a modelagem, criação e manipulação de bancos de dados, aplicando os conceitos aprendidos em aula para o desenvolvimento de sistemas estruturados e organizados.

---

## 📚 Conteúdos Estudados

Durante a disciplina são trabalhados diversos conceitos fundamentais de banco de dados, como:

* Conceitos básicos de **Banco de Dados**
* **Modelagem de dados**
* **Modelo Entidade-Relacionamento (MER)**
* **Modelo Relacional**
* Criação de **tabelas**
* **Chaves primárias e estrangeiras**
* **Normalização de dados**
* Manipulação de dados com **SQL**
* Consultas utilizando `SELECT`
* Operações com `INSERT`, `UPDATE` e `DELETE`

---

## 🗂 Estrutura do Repositório

O repositório pode conter:

* Exercícios práticos
* Modelagens de banco de dados
* Scripts SQL
* Projetos desenvolvidos em aula
* Atividades avaliativas

Cada pasta representa uma atividade ou conteúdo específico da disciplina.

---

## 🛠 Tecnologias Utilizadas

* **SQL**
* **SGBD (Sistema de Gerenciamento de Banco de Dados)**
* **MySQL / SQL Server** *(dependendo da atividade)*
* Ferramentas de modelagem de banco de dados

---

## 🎯 Objetivo do Repositório

* Consolidar conhecimentos em **Banco de Dados**
* Praticar **modelagem e consultas SQL**
* Registrar a evolução durante a disciplina
* Servir como material de estudo e consulta futura
